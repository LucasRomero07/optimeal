# Optimeal 🍽️

**Optimeal** es una aplicación web que te ayuda a encontrar recetas rápidas y fáciles según los ingredientes que tienes en tu despensa.

## 🚀 ¿Qué hace Optimeal?
- Selecciona los ingredientes que tienes disponibles.
- Filtra recetas por tiempo de cocción.
- Obtén recomendaciones de recetas que puedes preparar.
- Consulta el detalle de cada receta, ingredientes y pasos.
- Omite recetas que no te interesan.

## 📦 Instalación y uso

1. **Clona el repositorio:**
   ```bash
   git clone https://github.com/LucasRomero07/optimeal.git
   cd optimeal
   ```
2. **Instala las dependencias:**
   ```bash
   npm install
   ```
3. **Inicia la aplicación en modo desarrollo:**
   ```bash
   npm run dev
   ```
4. Abre tu navegador en [http://localhost:5173](http://localhost:5173)

## 🛠️ Tecnologías utilizadas
- React
- Vite
- JavaScript (ES6+)
- CSS

## 📁 Estructura principal
- `src/components/` — Componentes principales de la app (selector de despensa, lista y detalle de recetas).
- `src/data/` — Datos de ingredientes y recetas.
- `src/hooks/` — Hooks personalizados para almacenamiento local y recomendaciones.
